> The earliest days of web design

Back when the work meant slicing a designer’s static Photoshop layout into HTML, the tooling shifted constantly and the workflow followed every major design app of its era.

Today the latte-drinking designer crowd typically delivers Figma files, though Sketch and other vector tools serve the same purpose. Adobe’s counterpart in this space is Adobe XD.

All of them need to be supported, but the output that matters now is SVG. Scalable vector graphics are the only format worth caring about, and there are solid technical reasons for that.
